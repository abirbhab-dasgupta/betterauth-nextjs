# AuthNext

> [!IMPORTANT]
> **Disclaimer**: This project is under active development and is considered a work in progress. Features and functionality are subject to change.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Authentication**: Better Auth
- **Database**: Neon (PostgreSQL)
- **ORM**: Drizzle ORM
- **Styling**: Tailwind CSS 4
